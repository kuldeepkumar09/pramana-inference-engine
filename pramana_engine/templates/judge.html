How To Use

  1. Upload a valid JSON dataset (list of inference payloads).
  2. Click Run Judge Analysis to compute quality metrics.
  3. Review the R1-R5 compliance checklist, pramana distribution, and per-row detail table.
  4. Click Detail on any row to inspect the full epistemic trace.
Accepted: UTF-8 JSON — list of inference payloads or single payload with rules/evidence/request.
