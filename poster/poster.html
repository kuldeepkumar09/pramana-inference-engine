Pramana Inference Engine: Implementing Nyāya Epistemology as an AI Reasoning Framework

A RAG + LLM System Classifying Inferences as Valid, Unjustified, Suspended, or Invalid
IIT Delhi · Department of Computer Science & Engineering · 2026
CONF-2026 · Unriddling Inference · Poster Session
We present the Pramana Inference Engine — a neuro-symbolic AI system grounded in Nyāya philosophy’s प्रमाण (pramāṇa) theory. Five epistemic sources — pratyākṣa, anumāna, śabda, upamāna, arthāpatti — serve as weighted evidence channels in a hybrid RAG pipeline. A formal epistemic evaluator enforces five structural requirements (R1–R5), producing machine-readable verdicts (VALID / UNJUSTIFIED / SUSPENDED / INVALID) with full inference traces. 120 / 120 tests pass. Deployed as a Flask REST API with a 4-tab unified UI.
1 · Introduction

The Problem: Modern AI systems produce confident answers without distinguishing logically valid from epistemically justified conclusions — a critical gap for high-stakes reasoning.

Nyāya Solution न्याय (c. 200 BCE): India’s most systematic inference theory requires every claim to pass both a logical gate and an epistemic gate before acceptance as pramā (true cognition).

# Nyāya Aumāna — Five-Member Inference Paksha "the hill" # subject Sadhya "has fire" # predicate to prove Hetu "has smoke" # inferential mark Vyapti "smoke → fire" # invariable relation
Epistemic Requirements (R1–R5)
R1Represent propositions and evidence programmatically R2Explicitly check justification before accepting R3Distinguish VALID / UNJUSTIFIED / SUSPENDED R4Reject invalid inference patterns (fallacies) R5Output machine-readable reasoning traces

✓ VALID
Logic sound + epistemic warrant

No pramāṇa support
⏸ SUSPENDED
Defeater present — saṃśaya
✗ INVALID
Logical fallacy detected

Nyāya’s saṃśaya संशय (doubt) maps directly to SUSPENDED: a defeater (bādhita बाधित) suspends judgment pending further evidence.

4 · Discussion

Nyāya’s three-gate requirement (vyāpti holds + hetu grounded + no bādhita) maps precisely onto R4 (pattern), R2 (justification), and the SUSPENDED status.

Nyāya: bādhita बाधित
Western: Undercutting defeater (Pollock 1987)
Nyāya: saṃśaya संशय
Western: Epistemic suspension (Pyrrho)
